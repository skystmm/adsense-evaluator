# 前端页面演示指南

## 快速开始

### 1. 启动后端服务
```bash
cd backend
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 启动前端服务
```bash
cd frontend
npm run dev
```

### 3. 访问应用
打开浏览器访问：http://localhost:3000

---

## 页面演示流程

### 场景 1：查看评估报告详情

1. **访问首页** (http://localhost:3000)
   - 输入测试 URL：`https://example.com`
   - 勾选"AI 分析建议"
   - 点击"开始评估"

2. **查看报告** (自动跳转到 `/report/[id]`)
   - 查看顶部综合评分仪表盘
   - 查看雷达图展示各维度得分
   - 滚动查看问题清单（按严重程度排序）
   - 查看 AI 智能建议卡片
   - 点击"导出报告"测试打印功能
   - 点击"分享"测试链接复制

3. **关键特性展示**
   - ✅ 分数颜色自动变化（绿/黄/橙/红）
   - ✅ 问题优先级标识（严重/中等/轻微）
   - ✅ 响应式布局（调整浏览器窗口大小）
   - ✅ 流畅的加载动画

---

### 场景 2：查看历史记录

1. **访问历史记录页面**
   - 点击导航栏"历史记录"或访问 `/history`

2. **搜索功能**
   - 在搜索框输入：`example`
   - 查看实时过滤结果
   - 点击 X 清除搜索

3. **筛选功能**
   - 点击右上角"筛选"按钮
   - 选择"时间范围"：最近 7 天
   - 选择"最低分数"：60 分以上
   - 点击"应用筛选"
   - 点击"清除"重置筛选条件

4. **分页和删除**
   - 查看底部的分页控件
   - 点击"下一页"（如果有更多数据）
   - 点击某个记录跳转到报告详情
   - 点击垃圾桶图标删除记录（带确认提示）

---

### 场景 3：查看统计分析

1. **访问统计页面**
   - 点击导航栏"统计"或访问 `/stats`

2. **时间范围切换**
   - 点击右上角"最近 7 天"、"最近 30 天"、"最近 90 天"
   - 观察图表数据动态刷新

3. **查看各类图表**
   - **评分分布饼图**: 查看各分数段占比
   - **评分趋势折线图**: 查看平均分变化趋势
   - **维度雷达图**: 查看 4 个维度的平均分
   - **常见问题 TOP 10**: 查看最常见的问题
   - **评估数量趋势**: 查看每日评估数量

4. **交互功能**
   - 鼠标悬停图表查看详细信息
   - 观察趋势箭头的颜色变化（绿/红）

## 常见问题排查

### 问题 1：页面显示"加载失败"
**原因**: 后端服务未启动
**解决**: 确保后端运行在 http://localhost:8000

### 问题 2：图表不显示
**原因**: Recharts 数据格式问题
**解决**: 检查 API 返回的数据格式，确保包含必要的字段

### 问题 3：样式错乱
**原因**: TailwindCSS 未正确加载
**解决**: 检查 `globals.css` 是否正确引入，清除浏览器缓存

### 问题 4：路由 404
**原因**: Next.js 路由配置问题
**解决**: 确保使用 App Router 结构，文件路径正确

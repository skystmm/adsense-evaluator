# Google AdSense 评估网站 - UI 设计文档

## 1. 设计风格

### 1.1 整体风格
- **现代简洁**: 清晰的视觉层次，留白充足
- **专业可信**: 使用蓝色系为主色调，传达专业感
- **数据可视化**: 图表清晰，评分直观

### 1.2 配色方案
- **主色**: #2563EB (蓝色)
- **成功色**: #10B981 (绿色)
- **警告色**: #F59E0B (橙色)
- **危险色**: #EF4444 (红色)
- **中性色**: #6B7280 (灰色)

### 1.3 字体
- **标题**: Inter / System UI
- **正文**: Inter / System UI
- **代码**: JetBrains Mono / Monospace

### 2.3 历史记录页 (History Page)

#### 功能
- 显示用户历史评估记录
- 支持搜索和筛选
- 可查看详细报告
- 支持删除记录

## 3. 组件设计

### 3.1 评分组件
- 环形进度条显示总分
- 颜色根据分数变化（红→橙→绿）
- 动画效果

### 3.2 指标卡片
- 图标 + 标题 + 分数
- 悬停显示详细说明
- 响应式布局

### 3.3 问题列表
- 优先级标签（高/中/低）
- 问题描述
- 整改建议
- 展开/收起详情

### 3.4 加载状态
- 骨架屏
- 进度指示器
- 阶段性提示（爬取中→分析中→生成报告）

## 4. 响应式设计

### 断点
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### 适配策略
- 移动端：单列布局
- 平板：双列布局
- 桌面：多列布局

## 5. 交互细节

### 5.1 动画
- 页面切换：淡入淡出
- 评分显示：数字滚动 + 进度条动画
- 列表项：滑入动画

### 5.2 反馈
- 按钮点击：微缩放
- 表单验证：实时提示
- 成功/错误：Toast 通知

### 5.3 无障碍
- 键盘导航支持
- 屏幕阅读器友好
- 颜色对比度符合 WCAG 2.1 AA
